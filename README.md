# 🧱 Explicación paso a paso del HTML 

## 1. Qué es HTML

HTML es el lenguaje que se usa para crear la estructura de una página web.

Con HTML podemos añadir:

- Títulos
- Textos
- Imágenes
- Enlaces
- Tablas
- Formularios
- Secciones o cajas

HTML pone el contenido y la estructura y CSS da el estilo, 

---

## 2. Inicio del documento

<!DOCTYPE html>
<html lang="es">

🧠 Explicación:

<!DOCTYPE html> → indica que el documento es una página HTML5
<html> → es la etiqueta principal que contiene toda la web
lang="es" → indica que el idioma de la página es español

💡Idea fácil de recordar:

Aquí le estás diciendo al navegador: “voy a empezar una página web en español”.

## 3. Parte <head>: configuración de la página
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>NAYA</title>
  <link rel="stylesheet" href="style.css">
</head>

🧠 Explicación:

<head> → contiene información de configuración, no contenido visible principal
<meta charset="UTF-8"> → permite que se vean bien las tildes, la ñ y caracteres especiales
<meta name="viewport" ...> → hace que la web se adapte mejor a móviles y pantallas pequeñas
<title>NAYA</title> → nombre que aparece en la pestaña del navegador
<link rel="stylesheet" href="style.css"> → conecta el HTML con el archivo CSS

💡Idea fácil de recordar:

El head es como la parte técnica y de preparación de la página.

## 4. Parte <body>: contenido visible

<body>
Explicación:
<body> contiene todo lo que el usuario va a ver en la web

💡Idea fácil de recordar:

Todo lo visible de la página va dentro del body.

## 5. Título o frase principal

<h4 class="texto_plano">Construyendo una identidad digital.</h4>

🧠Explicación:

<h4> → es un encabezado o título
class="texto_plano" → esa clase sirve para aplicar estilos desde CSS

💡Idea fácil de recordar:

Aquí estás mostrando una frase destacada en la página.

## 6. Caja o contenedor de texto

<div class="card_textoplano">
Explicación:
<div> → crea una caja o bloque para agrupar contenido
class="card_textoplano" → esa caja tendrá estilos definidos en CSS

💡Idea fácil de recordar:

Un div sirve para organizar partes de la web en bloques.

## 7. Párrafos de presentación

<p class="texto_plano">Transformo mis experiencias y lo que observo del mundo en música e historias.</p>
<p class="texto_plano">Estas viendo sólo el comienzo...</p>

🧠 Explicación:
<p> → crea un párrafo de texto
class="texto_plano" → aplica el estilo CSS a esos textos

💡Idea fácil de recordar:
Aquí estoy contando quién es Naya y qué transmite.

## 8. Sección de YouTube

<div class="youtube">

🧠 Explicación:
este div agrupa el enlace y el icono relacionados con YouTube
Idea fácil de recordar:

Sirve para separar esta parte de otras secciones de la web.

## 9. Enlace al primer sencillo

<a href="https://www.youtube.com/watch?v=cI5OJ3hF7io" target="_blank" class="boton">Primer sencillo</a>

🧠 Explicación:
<a> → crea un enlace
href="..." → indica la dirección a la que lleva el enlace
target="_blank" → abre el enlace en una pestaña nueva
class="boton" → permite darle estilo visual con CSS

💡Idea fácil de recordar:
Este enlace funciona como un botón que lleva al vídeo de YouTube.

## 10. Salto de línea

<br>

🧠 Explicación:
<br> → hace un salto de línea

💡Idea fácil de recordar:
Es como bajar a la siguiente línea sin empezar una sección nueva.

## 11. Imagen del icono de YouTube

<img src="youtube.png" alt="YouTube" class="icono">

🧠 Explicación:
<img> → muestra una imagen
src="youtube.png" → indica qué archivo de imagen se va a mostrar
alt="YouTube" → texto alternativo por si la imagen no carga
class="icono" → aplica estilo desde CSS

💡Idea fácil de recordar:

Aquí estás mostrando el icono visual de YouTube.

## 12. Sección de TikTok
<div class="tiktok">

🧠 Explicación:
este div agrupa el enlace y el icono relacionados con TikTok

💡Idea fácil de recordar:

Sirve para tener organizada la parte de TikTok por separado.

## 13. Enlace a TikTok

🧠Explicación:
<a> → crea un enlace
href → indica a dónde lleva

class="boton2" → permite darle estilo con CSS

## 14. Imagen del icono de TikTok

<img src="tiktok.png" alt="tiktok" class="icono2">

🧠 Explicación:

💡Idea fácil de recordar:
Es el apoyo visual del enlace de TikTok.

## 15. Cierre de etiquetas

Al final del archivo debes cerrar correctamente todas las etiquetas abiertas.
Ejemplo:

</div>
</div>
</div>
</body>
</html>

🧠Explicación:
Cada div que abres debe cerrarse, imaginémonos la referencia de un sandwich, siempre un pan arriba y un pan abajo.
Cambién deben cerrarse body y html

💡Idea fácil de recordar:
Si abres una caja, luego tienes que cerrarla.

# 🎨 Explicación paso a paso del CSS 

## 1. Qué es CSS

CSS es el lenguaje que se usa para dar estilo a una página web.

Con CSS podemos cambiar:

- colores
- tamaños
- fondos
- bordes
- posición de elementos
- aspecto visual general

HTML crea la estructura y CSS la parte visual.

---

## 2. Fondo de la página


body {
  background-image: url(./nayabackground.png);
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}

🧠 Explicación:
body → representa todo el cuerpo visible de la página
background-image → pone una imagen de fondo
url(./nayabackground.png) → indica la ruta de la imagen
background-size: cover → hace que la imagen cubra toda la pantalla
background-position: center → centra la imagen
background-repeat: no-repeat → evita que la imagen se repita varias veces

💡 Idea fácil de recordar:

 Este bloque sirve para poner una imagen grande de fondo y hacer que se vea bien ocupando toda la página

## 3. Color de texto

.texto_plano {
  color: aliceblue;
}

🧠 Explicación:
.texto_plano → selecciona todos los elementos que tengan esa clase
color → cambia el color del texto
aliceblue → es un color muy claro, casi blanco

💡 Idea fácil de recordar:
Este bloque sirve para que el texto se vea claro sobre un fondo oscuro.

## 4. Tarjeta o caja con efecto visual

.card_textoplano {
  backdrop-filter: blur(10px);
  border-radius: 10px;
}

🧠 Explicación:
.card_textoplano → selecciona la caja que contiene el texto
backdrop-filter: blur(10px) → desenfoca lo que hay detrás de la caja
border-radius: 10px → redondea las esquinas

💡 Idea fácil de recordar:
Este bloque hace que la caja tenga un estilo más moderno, como un cristal desenfocado.

## 5. Color del formulario 

.formulario {
  color: blueviolet;
}

🧠 Explicación:
.formulario → selecciona la parte del formulario
color: blueviolet → cambia el color del texto a violeta

💡 Idea fácil de recordar:
Sirve para dar identidad visual al formulario y que combine con la estética de la web.

## 6. Tamaño del icono de YouTube

.icono {
  width: 100px;
  height: 60px;
  vertical-align: middle;
  }
🧠 Explicación:
.icono → selecciona la imagen del icono
width → ancho de la imagen
height → alto de la imagen
vertical-align: middle → ayuda a alinear la imagen con el texto

💡 Idea fácil de recordar:
Sirve para controlar el tamaño y la alineación del icono.

7. Tamaño del icono de TikTok
.icono2 {
  width: 100px;
  height: 60px;
  vertical-align: middle;
}

🧠 Explicación: Hace lo mismo que el bloque anterior, pero para el segundo icono.
Permite que ambos iconos tengan un tamaño parecido y se vean ordenados.
